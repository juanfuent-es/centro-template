Mapa de Ruta · Bloque 1

Primer bloque del curso: Fundamentos Web y Herramientas para Comunicación Visual (7º semestre). Dos sesiones por semana, enfoque práctico y publicable.

Sesiones del Bloque

  1. Introducción: La web como plataforma de narrativa visual interactiva
  2. Terminal, Git & GitHub: repositorios, commits, ramas, Pages
  3. Markdown: README, estructura y tipografía para documentación
  4. HTML semántico: jerarquía, secciones, navegación
  5. Layout reutilizable: _layouts, _includes, componentes y grid (Skeleton)
  6. Diseño visual en CSS: paleta de tránsito CDMX, tipografías, responsive
  7. Accesibilidad y performance: contraste, alt, metadatos, favicons
  8. Publicación: GitHub Pages, pruebas y revisión final

Entregables del Bloque

  • Sitio base en GitHub Pages con layout reutilizable
  • README con brief y estructura del proyecto
  • Secciones mínimas: Home, Guía, Recursos/Referencias
  • Guía de estilo con componentes y paleta
  • Accesibilidad mínima: alt, títulos, contraste AA
  • Despliegue configurado y funcional

Señalamientos · Criterios de Evaluación

Evaluación del primer bloque basada en calidad técnica, claridad visual y despliegue.

Rúbrica (resumen)

  • Estructura semántica (20%) – uso correcto de header/nav/main/section/footer
  • Layout reutilizable (20%) – _layouts, _includes y componentes consistentes
  • Diseño y sistema (20%) – paleta tránsito, tipografías, escala, espaciados
  • Accesibilidad y contenidos (15%) – contraste, alt, jerarquía textual
  • Repositorio y README (15%) – commits claros, documentación, licencias
  • Despliegue (10%) – sitio público en Pages con metadatos y favicon

Checklist rápido

  • Commit inicial + README con brief
  • Home con hero y navegación clara
  • Guía de estilo con botones, títulos, grid
  • Paleta aplicada (naranja metro, rosa, verde, rojo)
  • Contrast checker y alt en imágenes
  • GitHub Pages activo con dominio del repo
