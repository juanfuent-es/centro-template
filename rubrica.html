Reglamento de Evaluación

Esta rúbrica evalúa diferentes aspectos del desarrollo web moderno. Cada criterio se evalúa en una escala de 1-4 puntos.

Sistema de puntuación

Esta rúbrica evalúa diferentes aspectos del desarrollo web moderno. Cada criterio se evalúa en una escala de 1-4 puntos, donde 4 es excelente y 1 necesita mejora.

1. Documentación del Vehículo (4 puntos)

Evaluación de favicon, OpenGraph, título y estilos. Se valora la implementación correcta de metadatos y la organización del código.

Excelente (4 puntos)

  • Favicon personalizado implementado correctamente
  • Metadatos OpenGraph completos (title, description, image, url)
  • Título de página descriptivo y único
  • Metadatos SEO optimizados (keywords, author, viewport)
  • Estructura de estilos bien organizada

Bueno (3 puntos)

  • Favicon implementado
  • Metadatos OpenGraph básicos presentes
  • Título de página apropiado
  • Algunos metadatos SEO incluidos
  • Estilos organizados

Satisfactorio (2 puntos)

  • Favicon básico o por defecto
  • Metadatos OpenGraph mínimos
  • Título genérico
  • Metadatos SEO limitados
  • Estilos básicos

Necesita Mejora (1 punto)

  • Sin favicon
  • Metadatos OpenGraph ausentes o incorrectos
  • Sin título o título inapropiado
  • Sin metadatos SEO
  • Estilos desorganizados o ausentes

2. Carga y Mercancía (4 puntos)

Evaluación de la calidad y estructura del contenido. Se valora la relevancia, organización y accesibilidad de la información presentada.

Excelente (4 puntos)

  • Contenido relevante y bien estructurado
  • Jerarquía de información clara
  • Contenido accesible y comprensible
  • Información actualizada y precisa
  • Contenido original y de calidad

Bueno (3 puntos)

  • Contenido relevante
  • Estructura de información clara
  • Contenido comprensible
  • Información precisa
  • Contenido de buena calidad

Satisfactorio (2 puntos)

  • Contenido básicamente relevante
  • Estructura de información aceptable
  • Contenido comprensible con esfuerzo
  • Información mayormente precisa
  • Contenido de calidad aceptable

Necesita Mejora (1 punto)

  • Contenido irrelevante o confuso
  • Estructura de información poco clara
  • Contenido difícil de comprender
  • Información imprecisa o desactualizada
  • Contenido de baja calidad

3. Señalización Vial (4 puntos)

Evaluación del uso correcto de elementos HTML5 semánticos. Se valora la estructura semántica y accesibilidad del código HTML.

Excelente (4 puntos)

  • Uso correcto de elementos HTML5 semánticos (header, nav, main, section, article, aside, footer)
  • Estructura jerárquica de encabezados (h1-h6) apropiada
  • Elementos de lista (ul, ol, li) utilizados correctamente
  • Etiquetas de texto semánticas (strong, em, mark, time)

Bueno (3 puntos)

  • Uso mayormente correcto de elementos semánticos
  • Jerarquía de encabezados apropiada
  • Elementos de lista utilizados correctamente
  • Algunas etiquetas semánticas implementadas

Satisfactorio (2 puntos)

  • Uso básico de elementos semánticos
  • Jerarquía de encabezados aceptable
  • Elementos de lista básicos
  • Pocas etiquetas semánticas

Necesita Mejora (1 punto)

  • Uso incorrecto o ausente de elementos semánticos
  • Jerarquía de encabezados incorrecta
  • Elementos de lista mal utilizados
  • Sin etiquetas semánticas

4. Adaptabilidad de Vía (4 puntos)

Evaluación de la adaptabilidad a diferentes dispositivos

Excelente (4 puntos)

  • Diseño completamente responsive en todos los dispositivos
  • Media queries implementadas correctamente
  • Imágenes y contenido escalables
  • Navegación adaptativa
  • Tipografía legible en todos los tamaños

Bueno (3 puntos)

  • Diseño responsive en la mayoría de dispositivos
  • Media queries implementadas
  • Contenido mayormente escalable
  • Navegación funcional en diferentes tamaños
  • Tipografía legible

Satisfactorio (2 puntos)

  • Diseño responsive básico
  • Algunas media queries implementadas
  • Contenido parcialmente escalable
  • Navegación funcional en tamaños principales
  • Tipografía mayormente legible

Necesita Mejora (1 punto)

  • Diseño no responsive o con problemas
  • Media queries ausentes o incorrectas
  • Contenido no escalable
  • Navegación problemática en diferentes tamaños
  • Tipografía ilegible en algunos dispositivos

5. Pintura y Señalización (4 puntos)

Evaluación de la paleta de colores y coherencia visual

Excelente (4 puntos)

  • Paleta de colores coherente y profesional
  • Contraste adecuado para accesibilidad
  • Uso consistente de colores en toda la aplicación
  • Colores que reflejan la identidad del proyecto
  • Variables CSS para fácil mantenimiento

Bueno (3 puntos)

  • Paleta de colores coherente
  • Contraste aceptable
  • Uso mayormente consistente de colores
  • Colores apropiados para el proyecto
  • Algunas variables CSS implementadas

Satisfactorio (2 puntos)

  • Paleta de colores básica
  • Contraste mínimo aceptable
  • Uso parcialmente consistente de colores
  • Colores básicamente apropiados
  • Variables CSS limitadas

Necesita Mejora (1 punto)

  • Paleta de colores incoherente
  • Contraste inadecuado
  • Uso inconsistente de colores
  • Colores inapropiados
  • Sin variables CSS

6. Partes y Repuestos (4 puntos)

Evaluación de la organización y reutilización del código

Excelente (4 puntos)

  • Componentes reutilizables y modulares
  • Código bien organizado y documentado
  • Separación clara de responsabilidades
  • Componentes accesibles y semánticos
  • Uso de mejores prácticas de desarrollo

Bueno (3 puntos)

  • Componentes mayormente reutilizables
  • Código organizado
  • Separación de responsabilidades clara
  • Componentes accesibles
  • Buenas prácticas implementadas

Satisfactorio (2 puntos)

  • Componentes básicamente reutilizables
  • Código organizado básicamente
  • Separación de responsabilidades aceptable
  • Componentes mayormente accesibles
  • Algunas buenas prácticas implementadas

Necesita Mejora (1 punto)

  • Componentes no reutilizables
  • Código desorganizado
  • Separación de responsabilidades poco clara
  • Componentes no accesibles
  • Pocas buenas prácticas implementadas

Escala de Multas

Sistema de puntuación y calificación final

Calificaciones Altas

  • 24-22 puntos: Excelente 10
  • 21-18 puntos: Muy Bueno 9
  • 17-14 puntos: Bueno 8

Calificaciones Regulares

  • 13-10 puntos: Satisfactorio 7
  • 9-6 puntos: Necesita Mejora 6
  • 5-0 puntos: Insuficiente 5
